方案对比

A 双环平台

  • 直观表达"承载"语义
  • 与现有限制连线风格差异大，一眼可辨
  • 平台宽度可随距离缩放

A 的考量

  • 实现复杂度较高
  • 两个光环 + 平台占用视觉空间大

B 锁链牵引

  • 复用现有连线渲染逻辑
  • 实现最简单
  • 链子动画有动态感

B 的考量

  • 与"压制连线"容易混淆
  • "锁链"语义偏向控制/负面

C 光环包裹

  • 语义贴合"护盾保护"
  • 视觉简洁，不杂乱
  • 不同阵营可用不同颜色

C 的考量

  • 光环可能与草丛范围混淆
